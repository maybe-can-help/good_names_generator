# good_names_generator

## How to compile
Нужно установить `Python`

### Linux
```bash
python3 -m venv venv
source venv/bin/activate
pip3 install pyinstaller
pyinstaller --onefile src/good_names_generator.py
```

### Windows
```console
python -m venv venv
source venv/vin/activate
pip install pyinstaller
pyinstaller --onefile src\good_names_generator.py
```
